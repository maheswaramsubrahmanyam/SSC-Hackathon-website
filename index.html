Event Structure
  • Introduction and Kickoff (2:00 PM - 2:30 PM): Briefing on the rules, objectives, and project expectations.
  • Hackathon (2:30 PM - 7:30 PM): Participants work on their projects. Mentors will be available for guidance.
  • Submission (7:30 PM - 8:00 PM): Final submission of projects.

How to Participate
  • Registration: Participants must register online through the provided link.
  • Team Formation: Teams can consist of 2-4 members from the same or different branches.
  • Submission Format: Projects must be submitted in the prescribed format, including source code, documentation, and a presentation.

CSE HACKATHON
PROBLEM STATEMENTS

Computer Science Engineering (CSE)

  • Decentralized Identity Verification System: Develop a blockchain-based identity verification system to ensure secure and tamper-proof identity management.
  • Real-time Cyber Threat Detection: Create an AI-powered system that monitors network traffic in real-time to detect and mitigate cyber threats.
  • Automated Code Review Tool: Build a machine learning-based tool that automatically reviews code for potential bugs, vulnerabilities, and style inconsistencies.
  • Smart Traffic Management System: Design a real-time traffic management system using IoT and AI to optimize traffic flow and reduce congestion.
  • Distributed Cloud Storage System: Develop a decentralized cloud storage system that ensures data redundancy, security, and accessibility.
  • AI-Driven Customer Support Chatbot: Create an AI chatbot capable of understanding and resolving customer queries in real-time across various platforms.
  • Predictive Maintenance for Industrial Equipment: Design a predictive maintenance system using IoT sensors and machine learning to predict equipment failures before they occur.
  • Personalized Learning Platform: Develop an adaptive learning platform that customizes educational content based on the learner's pace and performance.
  • Virtual Reality Collaboration Tool: Build a VR-based tool that allows remote teams to collaborate in a virtual environment for enhanced productivity.
  • Blockchain-Based Voting System: Create a secure and transparent voting system using blockchain technology to ensure the integrity of the voting process.
